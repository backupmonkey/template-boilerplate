This directory is used to implement custom fonts (either Google Webfonts or other) into your template.

How to use custom fonts:

1. Add the fonts you want to use in your template to this directory.

2. Use the @font-face rule to specify custom fonts you want to use in your template.
Learn more at MDN web docs: https://developer.mozilla.org/en-US/docs/Web/CSS/@font-face

3. Feel Monkey!
